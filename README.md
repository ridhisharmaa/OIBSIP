# Oasis Infobyte Data Science Internship (OIBSIP)

This repository contains the projects completed as part of my **Data Science Internship** at **Oasis Infobyte (OIBSIP)**. Each project focuses on applying data science and machine learning concepts to solve real-world problems using Python and popular data science libraries.

## Projects

### 1. Iris Flower Classification
- Built a machine learning model to classify Iris flower species.
- Performed data preprocessing, exploratory data analysis (EDA), model training, and evaluation.

**Tech Stack:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

---

### 2. Unemployment Analysis with Python
- Analyzed unemployment trends in India using real-world datasets.
- Conducted data cleaning, visualization, and statistical analysis to derive meaningful insights.

**Tech Stack:** Python, Pandas, NumPy, Matplotlib, Seaborn

---

### 3. Car Price Prediction
- Developed a regression model to predict the selling price of used cars based on multiple features.
- Included data preprocessing, feature engineering, model training, and performance evaluation.

**Tech Stack:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
